# <a name="suppressreadreceipt"></a>SuppressReadReceipt

**SuppressReadReceipt 要素** は、読み取り受信を抑制するために使用されます。 
  
```xml
<SuppressReadReceipt>
   <ReferenceItemId/>
</SuppressReadReceipt>
```

 **SuppressReadReceiptType**
