# <a name="mailboxmoveenabled-soap"></a>MailboxMoveEnabled (SOAP)

**MailBoxMoveEnabled 要素** は **MailboxMoveEnabled() フラグを表** します。 **MailBoxMoveEnabled 要素** は内部でのみ使用できます。 この要素は、クライアントでは使用されません。 
  
```XML
<MailBoxMoveEnabled>true | false</MailBoxMoveEnabled>
```

**Boolean**
