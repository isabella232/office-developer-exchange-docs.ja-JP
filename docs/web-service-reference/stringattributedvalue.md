# <a name="stringattributedvalue"></a>StringAttributedValue

**StringAttributedValue** 要素は、ペルシャ要素に関連付けられた属性の配列内のインスタンスを指定します。 
  
```XML
<StringAttributedValue>
   <Value/>
   <Attributions/>
</StringAttributedValue>
```

 **StringAttributedValueType**
