# <a name="phoneticfirstname"></a>PhoneticFirstName

**PhoneticFirstName** 要素には、連絡先の最初の名前が含まれている。スペルは電話で入力されます。 
  
```XML
<PhoneticFirstName/>
```

 **string**
## <a name="attributes-and-elements"></a>属性と要素

以下のセクションで、属性、子要素、親要素について説明します。
  
### <a name="attributes"></a>属性

なし。
  
### <a name="child-elements"></a>子要素

なし。
  
### <a name="parent-elements"></a>親要素

|**要素名**|**説明**|
|:-----|:-----|
|[Contact](contact.md) <br/> |ストア内の連絡先アイテムExchangeします。  <br/> |
   
## <a name="text-value"></a>テキスト値

テキスト値は、連絡先の最初の名前を表す文字列で、スペルは表形式です。
  
## <a name="remarks"></a>注釈

この要素を記述するスキーマは、Exchange Web サービスをホストする IIS 仮想ディレクトリに置かれています。
  
この要素は Exchange Server 2010 Service Pack 2 (SP2) で導入されました。
