# <a name="arrayofuserresponse-soap"></a>ArrayOfUserResponse (SOAP)

**ArrayOfUserResponse** 要素には [、UserResponse (SOAP) 要素の配列が含](userresponse-soap.md)まれています。 
  
```XML
<ArrayOfUserResponse>
   <UserResponse/>
</ArrayOfUserResponse>
```
