# <a name="numberofmemberswithnodata"></a>NumberOfMembersWithNoData

**NumberOfMembersWithNoData** 要素は、提案された会議時間と比較する空き時間データを公開していない配布リスト メンバーの数を表します。 この要素は、大きすぎる配布リストのメンバー、またはデータなし状態の **メンバーを表** します。

## <a name="remarks"></a>注釈

メールボックスを持つグループ内の連絡先は、予定表データを持つ配布リスト メンバーの例です。 連絡先には、次の **理由でデータ** なし状態が設定されている場合があります。 
  
- アクセス許可が不十分です。
    
- 配布リストが大きすぎて展開されません。
    
- Active Directory ディレクトリ サービスは使用できません。
    
この要素を記述するスキーマは、クライアント アクセス サーバーの役割がインストールされている MicrosoftExchange Server 2007 を実行しているコンピューターの EWS 仮想ディレクトリにあります。
