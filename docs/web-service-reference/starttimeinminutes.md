**StartTimeInMinutes** 要素は、メールボックス ユーザーの作業日の開始を表します。

### <a name="attributes"></a>属性

なし。

## <a name="text-value"></a>テキスト値

テキスト値は必須です。 テキスト値は、その日の開始から何分経過した作業日の開始を表します。 たとえば、開始時刻は 8 A.M です。 は 480 分で表されます。
  
この要素で使用できる値の範囲は 0 ~ 1440 です。
