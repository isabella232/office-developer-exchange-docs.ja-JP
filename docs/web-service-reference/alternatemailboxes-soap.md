# <a name="alternatemailboxes-soap"></a>AlternateMailboxes (SOAP)

**AlternateMailboxes 要素は**、代替メールボックスのコレクションを表します。 
  
```XML
<AlternateMailboxes>
   <AlternateMailbox/>
</AlternateMailboxes>
```

 **AlternateMailboxes**
