# <a name="protocolconnections-soap"></a>ProtocolConnections (SOAP)

**ProtocolConnections 要素には**、0 個以上のプロトコル接続が含まれる。 
  
```XML
<ProtocolConnections>
   <ProtocolConnection/>
</ProtocolConnections>
```

 **ProtocolConnections**
